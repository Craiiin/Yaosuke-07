# 布料物理模拟实验｜三种数值积分方法对比
**计算机图形学 - 实验 7**  
基于 Taichi 实现的质点-弹簧系统布料仿真，对比**显式欧拉 / 半隐式欧拉 / 隐式欧拉**积分器的稳定性、运动表现与数值特性。

---

## 项目信息
- **作者**：姚苏珂
- **开发环境**：Python 3.13 + Taichi 1.7.4
- **物理模型**：20×20 质点网格 + 4-邻域结构弹簧
- **固定约束**：布料左上角、右上角两点固定

## 核心功能
1. **结构弹簧布料模型**
   - 每个质点仅与**上下左右 4 个相邻质点**连接
   - 标准布料受力：重力、胡克弹簧力、速度阻尼力
2. **三种积分方法实时切换**
   - 显式欧拉（Explicit Euler）
   - 半隐式欧拉（Semi-Implicit Euler）
   - 简化隐式欧拉（Implicit Euler / 向后欧拉近似）
3. **可视化控制面板**
   - 积分器切换 / 暂停继续 / 重置仿真

## 交互说明
点击窗口右侧控制面板：
- `Explicit Euler`：切换至**显式欧拉**（不稳定、剧烈震荡）
- `Semi-Implicit`：切换至**半隐式欧拉**（稳定、自然弹性）
- `Implicit Euler`：切换至**隐式欧拉**（极稳定、强耗散）
- `Pause / Resume`：暂停 / 继续仿真
- `Reset`：重置布料初始状态

---

## 三种积分方法核心区别（实验重点）
### 1. 显式欧拉（Explicit Euler）
- **更新顺序**：先更新位置 → 后更新速度
- **原理**：使用**旧速度**更新位置
- **表现**：误差快速累积 → **剧烈震荡、质点抱团、极易发散爆炸**
- **特点**：计算快，但**稳定性极差**，不适合实际仿真

### 2. 半隐式欧拉（Semi-Implicit Euler）
- **更新顺序**：先更新速度 → 后更新位置
- **原理**：使用**新速度**更新位置
- **表现**：运动自然、下落迅猛、布料无过度拉伸、稳定回弹
- **特点**：**实时物理仿真行业标准**，平衡性能与稳定性

### 3. 隐式欧拉（Implicit Euler）
- **实现方式**：预测下一时刻位置 → 用**未来位置计算受力**
- **原理**：使用下一时刻的状态信息更新当前运动
- **表现**：极度稳定、运动柔和、能量快速耗散、无任何震荡
- **特点**：无条件稳定，但视觉效果偏“软、慢、粘”

## 实验结论
1. **显式欧拉**：计算简单但**数值不稳定**，能量持续增加，会出现异常震荡与发散。
2. **半隐式欧拉**：稳定性强、运动自然、无拉伸变形，是**布料模拟最优选择**。
3. **隐式欧拉**：无条件稳定，但会**过度耗散系统能量**，适合需要绝对稳定的场景。

本实验清晰展示了不同数值积分方法在质点-弹簧系统中的行为差异，可作为计算机图形学物理模拟基础实验的标准参考。
